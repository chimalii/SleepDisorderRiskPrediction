# SleepDisorderRiskPrediction
Gabriela Chimali Nava Ramírez | A01710530

**ABOUT** *Este repositorio contiene el código de un modelo de Aprendizaje Profundo (Deep Learning) diseñado para la clasificación multiclase del riesgo de trastornos del sueño. Utilizando un conjunto de datos de 100,000 registros calibrados con fuentes clínicas de alto nivel, se implementó un Perceptrón Multicapa (MLP) optimizado con técnicas de regularización y manejo de desbalance de clases, y una versión mejorada que incorpora conexiones residuales. Los resultados muestran que el modelo ResNet-MLP supera al MLP base, alcanzando una exactitud del 87.16% en el conjunto de prueba, junto con un F1-Score macro de 80.73% y una exactitud balanceada de 83.67%. Particularmente, se observa una mejora en la detección de casos clínicamente relevantes, como la clase "Severe", lo cual es crítico en aplicaciones médicas, superando incluso las limitaciones del desbalance estadístico inherente a estas. Estos hallazgos posicionan modelos de Deep Learning como una alternativa competitiva frente a enfoques tradicionales basados en árboles.*
